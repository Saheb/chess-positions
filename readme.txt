
SBT users:

1) sbt test - will run the tests
2) sbt run - will start the program
3) Enter the input parameters
4) You will see solutions and time taken on the console

From IDE:

1) Run the ChessPositions object
2) Enter the input parameters
3) You will see solutions and time taken on the console

P.S. Stats.txt contains stats for 1 use case.
Stats might vary running on unix terminals vs IDEs.
